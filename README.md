# 🍔 Food Meals UI  

A clean and modern **Flutter UI** for a food delivery app, showcasing beautiful design, responsive layouts, and smooth custom animations.  

---

## 🚀 Features  
- Fully **responsive design** for multiple screen sizes  
- **Custom animations** for an engaging user experience  
- **Clean and scalable folder structure** for easy maintenance  
- Support-ready for **multiple languages (i18n)**  
- **Clean architecture** implemented with **GetX**  
- **Reusable, customizable widgets** for rapid UI updates  
- Built following **SOLID principles** for better scalability and testability  

---

## 🛠 Tech Stack  
- **Flutter** — Cross-platform development  
- **GetX** — State management and routing  
- **Custom Painter** — For advanced UI elements  
- **Clean Architecture** — Organized, maintainable codebase  
- **SOLID Principles** — For clean and scalable development  
- **Custom Animations** — Smooth and high-performance transitions
